# 0x15. JavaScript - Web jQuery

In this, project we are learning to use jQuery to build interesting thing.
[jQuery] (https://jquery.com/) is a JavaScript framework that facilitated the
development with JavaScript.

** Styles**
The project contains many JavaScript scripts written with the semistandard
style.
