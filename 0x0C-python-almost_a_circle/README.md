# Python - Almost a circle
This project will review everything about Python :

+ Import
+ Exceptions
+ Class
+ Private attribute
+ Getter/Setter
+ Class method
+ Static method
+ Inheritance
+ Unittest
+ Read/Write file

And it also about :

+ args and kwargs
+ Serialization/Deserialization
+ JSON

## Project Structure
This project have two directories :

+ tests : for all unitests
+ models : for all modules
